Task alert

Easy configuration of alerts for data in Odoo that has a date field. Alerts (project tasks) are created using a period before the certain date is passed. This can be used on any Odoo object. Useful examples are task dead line, association membership end date and leaves in Odoo. Leaves can be managed using our module Absence management.

Define alert.

Create definition when and for whom a task is created. In this case a task is created every 14 days.
