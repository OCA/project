========================
Project Manager Invoicing
=========================

This module was written to help project managers in their invoicing tasks of
timesheet lines.

It allows to make detailed invoices. You can set a specific amount of hours
to invoice (e.g. for rounding to a quarter of an hour). You can set a product
to invoice (e.g. to invoice different type of service done by the same employee)


Installation
============

This modules requires hr_timesheet_invoice
